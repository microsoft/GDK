# UserList (JSON)
A collection of [User](json-user.md) objects. 
<a id="ID4ER"></a>

 
## UserList
 
The UserList object has the following specification.
 
| Member| Type| Description| 
| --- | --- | --- | 
| users| Array of [User (JSON)](json-user.md)| See JSON example below.| 
  
<a id="ID4EPB"></a>

 
## Sample JSON syntax
 

```json
{
    "users":
    [
        { "xuid":"12345" },
        { "xuid":"23456" }
    ] 
}
    
```
